# SwiftPath AI – Intelligent Emergency Vehicle Detection and Signal Optimization

## Project Overview
SwiftPath AI is an AI-based system designed to detect emergency vehicles such as ambulances and optimize traffic signals to provide them priority.

## Problem Statement
Emergency vehicles often face delays due to heavy traffic and fixed traffic signal timings. These delays can increase emergency response time.

## Proposed Solution
The system uses Artificial Intelligence and computer vision to detect an approaching emergency vehicle. Once detected, the traffic signal is optimized to give priority to the emergency vehicle.

## Technologies
- Python
- Artificial Intelligence
- Computer Vision
- YOLO Object Detection
- OpenCV
